## what can you expect?

a minimal, keyboard-first hugo theme forked from [paper-mod](https://adityatelange.github.io/hugo-PaperMod/) for a "whatever comes to my mind"-blog. i would like to thank my colleague [cstef](https://github.com/cestef) who set me up on static site generators, especially with inspo taken from [his blog](https://blog.cstef.dev/)

- single theme (dark only, sorry!)
- customized wherever i can to suit my personal needs
- centralized keyboard shortcuts in one partial
- vim-like status bar on content pages
- no build tooling beyond hugo assets pipeline
- a bunch of content if you like reading

## shortcuts

- `/` open search
- `?` toggle help
- `v` toggle vim status bar
- `r` open this repo
- `alt + c` toggle toc
- `alt + e` home
- `alt + p` posts
- `alt + u` top

## make this theme yours

1. clone this repo

```sh
git clone https://github.com/vahrina/dat.git
cd dat
```

2. become familiar with the hugo ecosystem: [docs](https://gohugo.io/about/introduction/) | visual learners: [fireship](https://www.youtube.com/watch?v=0RKpf3rK57I), [luke smith](https://www.youtube.com/watch?v=ZFL09qhKi5I)

feel free to adjust the theme in `/assets/css/extended/` & `/layouts/` to however you like it

> every theme will be slightly different to another, f.e. paper mod utilizes the `/assets/css/extended/` directory for styling
